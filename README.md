# dbk-share2
自由上传成人色情视频/图片/游戏/小说的成人网站盘点(长期更新，欢迎订阅)

几年前御三家（xvideos、pornhub、xhamster）因意外事件被锤，路人上传者们顺带躺枪，推特红迪汤不热等因不是专门平台，常随机夹号或限制发表长度；
其余一众上传站，或**不兼容真人和虚拟内容**、或**门可罗雀**、或**不支持搜亚洲语言**、或**审核层层加码**。

故发贴细数不错的上传站（为防不测，本帖不定期更新）

注：
1.作为无关政治正确的共识，请知晓**真人儿童色情内容**之法律风险。
2.任何色站都有随机移除上传内容之风险，只是程度差异，望知悉。
